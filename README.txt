# Build app with ts+webpack+yarn
[uk] 2018

1. Clone project from git
2. Install node (8.x.x)
3. Install npm latest: sudo npm install -g npm@latest
4. Install yarn: curl -o- -L https://yarnpkg.com/install.sh | bash
5. Run $ yarn install
6. Run $ yarn run start:server

Develop: Hot deployment to browser: http://localhost:8080/webpack-dev-server/

Try: change code in ./src/app/main/main.js

Extras:
Build app by: $ yarn run build:prod
Build create at: /dist
